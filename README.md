# Visualizing Housing Market Trends

## Project Description
This project analyzes housing market trends using Tableau and integrates the dashboard into a web application using Flask.

## Tools Used
- Tableau
- Python
- Flask

## Features
- Interactive Dashboard
- Story Visualization
- Data Filters
- Web Integration using Flask

## How to Run
1. Install Flask
2. Run:
   python app.py
3. Open:
   http://127.0.0.1:5000/
